# Tic Tac Toe Game in React

This is a simple Tic Tac Toe game built with React. The game allows two players to take turns marking the spaces in a 3Ã—3 grid with X and O. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game. If all nine squares are filled and neither player has three in a row, the game is a draw.

## Features

- Two-player mode
- Indicates the current player's turn
- Detects and announces the winner
- Detects and announces a draw
- Allows players to reset the game and play again
